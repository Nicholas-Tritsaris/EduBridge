# EduBridge

**EduBridge** — Connect. Learn. Grow.

EduBridge is a lightweight platform that connects students and teachers for assignments, messaging, and classroom collaboration. This repository contains the starter code and documentation for the EduBridge web app.

## Features
- **Classrooms** — create and manage class groups.
- **Assignments** — create, submit, and grade assignments.
- **Messaging** — teacher-student and peer communication.
- **Resource Hub** — upload and share learning materials.
- **Analytics** — basic engagement and performance insights.

## Tech stack
- Frontend: React (or your chosen framework)
- Backend: Node.js + Express (or your chosen backend)
- Database: PostgreSQL / MongoDB
- CI: GitHub Actions
